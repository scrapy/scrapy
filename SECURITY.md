# Security Policy

## Supported Versions

| Version   | Supported |
|-----------|-----------|
| 2.13.x    | ✅        |
| < 2.13.x  | ❌        |

## Reporting a Vulnerability

If you discover a security vulnerability in Scrapy, please [report](https://github.com/scrapy/scrapy/security/advisories/new) it responsibly using GitHub's advisory system.
