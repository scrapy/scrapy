.. _topics-email:

==============
Sending e-mail
==============

.. warning::

    The ``scrapy.mail`` module and the
    :class:`~scrapy.extensions.statsmailer.StatsMailer` extension are
    deprecated and will be removed in a future release. Please use a
    dedicated Python library such as `aiosmtplib
    <https://aiosmtplib.readthedocs.io/>`_ for sending emails.
